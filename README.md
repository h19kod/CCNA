# CCNA Network Project

## Description
This project was created using Cisco Packet Tracer as part of CCNA practice and learning. It demonstrates network design, device configuration, connectivity testing, and troubleshooting skills.

## Objectives
- Build a functional network topology
- Configure routers and switches
- Verify network connectivity
- Practice CCNA concepts
- Improve troubleshooting skills

## Technologies Used
- Cisco Packet Tracer
- Cisco Routers
- Cisco Switches
- Ethernet Networking

## Skills Demonstrated
- Network Design
- IP Addressing
- Router Configuration
- Switch Configuration
- VLANs
- DHCP
- Static Routing
- Network Troubleshooting

## Project Files

- network.pkt — Cisco Packet Tracer project file
- screenshots/ — Network topology screenshots
- README.md — Project documentation

## How to Open

1. Install Cisco Packet Tracer.
2. Download the project files.
3. Open the .pkt file using Packet Tracer.
4. Test connectivity and review configurations.

## Learning Outcomes

This project helped strengthen practical CCNA networking skills, including network setup, configuration, verification, and troubleshooting.
